## 📌 عنوان ووصف المشروع
**تتطوير منصة إدارة التكوين عبر الإنترنت: EduLive**

**EduLive** هي منصة ويب متكاملة وآمنة مخصصة لإدارة مراكز التكوين ومدارس اللغات. تهدف المنصة إلى رقمنة العمليات الإدارية والتعليمية وتسهيل التواصل بين الإدارة، الإطار التربوي، والطلبة من خلال:
* إدارة التسجيلات وقبول/رفض الطلبات مع إشعارات بريدية تلقائية.
* تنظيم الدورات، المستويات، والمجموعات الدراسية.
* مشاركة الموارد التعليمية (PDF، فيديوهات، ملفات Word/Excel).
* متابعة حضور الطلبة وتقدمهم الدراسي.
* نظام مراسلة داخلي يشمل المحادثات الجماعية والخاصة.
* لوحات تحكم متعددة مع إحصائيات تفصيلية ورسوم بيانية.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

### 1. الواجهة الأمامية (Frontend)
* **React 18 + TypeScript + Vite:** لبناء واجهات مستخدم تفاعلية وسريعة.
* **Tailwind CSS:** لتصميم واجهات متجاوبة وعصرية.
* **React Router v6 & Axios:** للتنقل وإرسال طلبات الـ HTTP.
* **Recharts:** لعرض الرسوم البيانية في لوحات التحكم.

### 2. الخادم الخلفي والقاعدة (Backend & DB)
* **Node.js + Express.js:** لبناء خادم التطبيق وواجهات الـ API.
* **MySQL:** قاعدة البيانات العلائقية لحفظ البيانات (educentre_db).
* **JWT (JSON Web Tokens) & Bcrypt:** لتأمين النظام والمصادقة المبنية على الأدوار.
* **Multer:** لمعالجة ورفع الملفات والمواد التعليمية بالسيرفر.
* **Nodemailer:** لإرسال رسائل البريد الإلكتروني التلقائية.

---

## 👥 أدوار المستخدمين في النظام
* 🌐 **زائر (Visitor):** تصفح الدورات والمستويات، وإرسال طلب تسجيل للمركز.
* 👑 **مدير (Admin):** تحكم كامل بالنظام، قبول أو رفض المسجلين، وإدارة الحسابات والإحصائيات.
* 👨‍🏫 **أستاذ (Teacher):** إدارة المجموعات الخاصة به، رفع المحاضرات، وتسجيل حضور وغياب الطلبة.
* 👨‍🎓 **طالب (Student):** متابعة الدروس والمجموعات، تحميل الملفات، ومتابعة جدول أوقاته ونظام حضورّه.
